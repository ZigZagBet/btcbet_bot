# Btcbet.cc bot by ZigZag

A bot for automatic BTC betting on server: [https://btcbet.cc](https://btcbet.cc/r/07RBg5)

### Prerequisites

python3 and higher
Tested on Python 3.6

### Executing
```
python btcbet_bot_by_ZigZag.py
or
./btcbet_bot_by_ZigZag.py
```
